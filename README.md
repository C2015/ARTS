# ARTS
ARTS 是 Algorithm、Review、Technique、Share 的简称，即每周至少做一个 leetcode 的算法题，阅读并点评至少一篇英文技术文章，学习至少一个技术技巧，至少分享一篇有观点和思考的技术文章。期待自己的蜕变。
